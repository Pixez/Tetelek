# Tetelek
A programozás Tételek a 11.a Osztálynak

Ha hivatkozni szeretnénk a Tételekre akkor használjuk az alábbi parancsot:
Tetel Tetelek = new Tetel();

innentől fogva lehet a tételeket használni az alábbi módon:
Teletek.

Ha bármi problémát találtok a kóddal vagy valami nem érthető akkor nyugottan írjakot üzenetet vagy e -mail -t nekem.
